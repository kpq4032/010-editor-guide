## 010 Editor 的核心功能值得单独说说

人们选 010 Editor 而不用其他十六进制编辑器，主要是以下几个理由。

**二进制模板技术**是最核心的差异点。普通十六进制编辑器只给你看一列列十六进制数，010 Editor 可以加载对应文件格式的模板（社区里已经有几百个现成的），自动把 ZIP、MP3、PNG、EXE 等格式解析成带字段名的树形结构。哪个字节对应文件头，哪个字节是压缩标志，一目了然。

**脚本自动化**用的是类 C 语法，写起来没有特别的门槛。批量修改几千个文件的某一段字节，写个脚本跑一遍，比手动操作快得多。

**大文件支持**是另一个实用点。内置的 DynaBin 数据引擎可以加载 4GB 以上的超大文件，磁盘镜像、固件包、内存转储这类东西直接打开，不会卡。

**磁盘和驱动器编辑**支持直接操作 NTFS、FAT32 等文件系统的扇区数据，对做数据恢复或底层调试的工程师来说很有用。

**数据分析工具**包含字节差异对比、直方图可视化、哈希校验算法等。逆向工程时分析两个版本的差异，或者验证文件完整性，都能直接在软件里做。
